# POJ的题目[Ants](http://poj.org/problem?id=1852)
解题关键在于理解两只蚂蚁的相遇对解题无影响
![相遇后会发生什么]({{ site.baseurl }}/images/algorithmmathInLife_1/ants.jpg)
```Java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        // 获取案例数
        int cases = scanner.nextInt();

        for (int i = 0; i < cases; i++) {
            // 获取杆长
            int poleLen = scanner.nextInt();
            // 获取蚂蚁数量
            int n = scanner.nextInt();
            // 根据蚂蚁数量创建蚂蚁位置数组
            int[] ants = new int[n];
            for (int j = 0; j < ants.length; j++) {
                ants[j] = scanner.nextInt();
            }

            // 计算最短时间和最长时间
            int minTime = 0, maxTime = 0;
            for (int j = 0; j < ants.length; j++) {
                // 无视两只蚂蚁相遇对解题无影响
                minTime = Math.max(minTime, Math.min(ants[j], poleLen - ants[j]));
                maxTime = Math.max(maxTime, Math.max(ants[j], poleLen - ants[j]));
            }
            // 输出结果
            System.out.println(minTime + " " + maxTime);
        }

        scanner.close();
    }
}
```
# 难度增加的抽签问题
##  二分搜索与$O({n^3}{\log{n}})$的算法
```Java
public class Main {
    public boolean solve(int target, int[] nums) {
        // 通过二分搜索改进的抽签问题
        for (int i : nums) {
            for (int j : nums) {
                for (int k : nums) {
                    int l = target - i - j - k;

                    // 二分搜索代码
                    int left = 0, right = nums.length - 1;
                    while (left <= right) {
                        int middle = (right - left) / 2 + left;
                        if (nums[middle] < target) {
                            left = middle + 1;
                        } else if (nums[middle] == target) {
                            return true;
                        } else {
                            right = middle - 1;
                        }
                    }
                }
            }
        }
        
        return false;
    }
}
```

## $O({n^2}{\log{n}})$的算法
先遍历一遍数组，将$k_c + k_d$保存到一个新的数组中，
再遍历一遍数组，使用二分搜索检查是否有$target - k_a -k_b$在新的数组中。
```Java
import java.util.Arrays;

public class Main {
    public boolean solve(int target, int[] nums) {
        // 保存k_c + k_d和的数组
        int[] newNums = new int[nums.length * nums.length];
        for (int i = 0; i < nums.length; i++) {
            for (int j = 0; j < nums.length; j++) {
                newNums[i * nums.length + j] = nums[i] + nums[j];
            }
        }

        // 对k_c + k_d数组进行排序
        Arrays.sort(newNums);

        for (int i : nums) {
            for (int j : nums) {
                // target-k_a-k_b
                int key = target - i- j;

                // 二分搜索代码
                int left = 0, right = nums.length;
                while (left < right) {
                    int middle = (right - left) / 2 + left;
                    if (newNums[middle] < key) {
                        left = middle + 1;
                    } else if (newNums[middle] == key) {
                        return true;
                    } else {
                        right = middle;
                    }
                }
            }
        }

        return false;
    }
}
```
